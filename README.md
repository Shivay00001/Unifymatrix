# UnifyMatrix-Starter

A 100% real open-source starter pack for an all-in-one communication app powered by Matrix with Telegram & WhatsApp bridging, login system using Firebase, and a frontend built in React.

## Features

- 🔐 Login & Signup using Firebase
- 💬 Chat using Matrix SDK
- 🔄 Telegram and WhatsApp bridging
- ⚙ Docker-based local deployment
- 🆓 Fully deployable on free Render
